## GIT常用命令

git最常用的命令共有6个分别是

```
git clone、git push、git add、git commit、git checkout、git pull]
```

当本地文件上传到Github上是需要先初始化仓库

```
git init
```

然后将文件添加进暂存区

```
git add .
```

通过

```
git commit
```

命令将暂存区内容添加到仓库中



##### 以下几条命令是其他几条git 常用命令

从远程仓库拷贝一个项目

```
git clone
```

将文件添加到缓存区

```
git add
```

比较暂存区和工作区的不同

```
git diff
```

使用外部差异工具比较文件的更改

```
git difftool
```

比较两个提交范围的差异

```
git rang-diff
```

回退版本

```
git reset
```

将文件从暂存区和工作区删除

```
git rm
```

移动或重命名工作区文件

```
git mv
```

添加注释

```
git notes
```



```

```



